ePayco Payment Acquirer

Highlights

  • After installation of this module you will see the ePayco in payment acquirers list.
  • Invoicing -> Configuration -> Payments -> Payment Acquirers
  • Configure ePayco for testing environment.

  • Now go to the website add product into the cart and go to the payment page.
  • We can see the option for ePayco payment.
  • Select the ePayco payment and click on pay now.

  • Once you will click on pay now button it will show you the popup for ePayco payment.
  • You have to enter card detail and click on pay.

  • Once your payment is done it will show you the payment successfull message.
